## Remaining Gate

To claim the whole goal is fully complete, run one external clean-machine test:

1. Copy `release\TeamsCaptionTranslator-v0.2.0` to a clean Windows user profile, Windows Sandbox, or VM.
   - For normal distribution, send `release\TeamsCaptionTranslator-v0.2.0.zip` and unzip it on the target machine.
2. Run:

   ```powershell
   powershell -NoProfile -ExecutionPolicy Bypass -File .\CLEAN_MACHINE_CHECK.ps1 -DeepSeekApiKey "sk-..."
   ```

3. Confirm `CLEAN_MACHINE_RESULT.txt` says `PASS`.
4. Also do one manual Teams run:
   - install by double-clicking `TeamsCaptionTranslatorSetup-0.2.0.exe`
   - launch from Start Menu
   - enter DeepSeek API key
   - click `Test DeepSeek`
   - click `Prepare OCR`
   - select the Teams subtitle region
   - click `Test OCR`
   - click `Start Translation`
   - confirm translated rolling captions appear and old content remains.
